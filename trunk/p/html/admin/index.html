Witti - Free layout for your CMS

Witti is free and simple admin layout which can be used by everyone who is going to create his own CMS or small administration system and doesn`t want wasting time with designing admin part of application. Witti includes all main parts that are used in majority of on-line administrations and after download is ready-to-use. You can see all parts in the left-side menu.

Witti is written in XHTML 1.0 Strict code cooperating with two CSS styles. First for on-screen media and print one. Of course, everything is valid.

Before using Witti read the licence please. After all I wish you lot of fun with programming your own admin system.
